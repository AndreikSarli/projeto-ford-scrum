# projeto-ford-scrum

Esse projeto foi feito como parte de um desafio sobre Git e metodologia Scrum.

A ideia foi praticar na prática como funciona o versionamento usando Git e a organização de tarefas dentro de um sprint.

## 🚀 O que foi feito

Nesse sprint eu trabalhei em duas tarefas:

* Alterar a cor dos nomes dos veículos no arquivo index.html (deixando em azul)
* Atualizar os textos dos links no arquivo contato.html

## 🌿 Organização

Usei branches pra separar as tarefas:

* `index` → alterações no index.html
* `contato` → alterações no contato.html

Depois fiz o merge de tudo na branch `main`, que ficou com a versão final.
